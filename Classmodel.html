E-03 Role Human Actor Permission profile assigned to Officers (Admin / Senior Officer / Analyst / Read-Only). Controls feature access. roleId, roleName, permissions[ ]
E-04 Report Core Domain Citizen submission about a suspected fake account. Entry point into the system; generates a Case upon AI confirmation. ticketNumber, platform, suspectUrl, reason, status, screenshots[ ]
E-05 Case Core Domain The central workflow entity representing one fake account investigation. Aggregates scan results, evidence, takedown requests and audit history. caseNumber, priority, status, fakeScore, slaDeadline, createdAt
E-06 SocialAccount Detection The suspected fake social media account being analysed. Stores scraped metadata used as ML features. username, platformUserId, followerCount, accountAge, postCount, isVerified
E-07 ScanResult Detection Output of the AI Detection Engine for one account. Stores composite FakeScore, per-pipeline sub-scores and SHAP explanations. fakeScore, metadataScore, imageScore, nlpScore, graphScore, confidence
E-08 Platform Infrastructure A registered social media platform (Facebook, Instagram, Twitter, YouTube). Stores API config, grievance email and compliance metrics. name, apiEndpoint, grievanceEmail, slaHours, complianceRate
E-09 SLAConfig Infrastructure SLA window rules per platform defined under IT Rules 2021 (36 hrs grievance / 72 hrs general). Controls reminder schedule. grievanceHours, generalHours, reminder50pct, reminder90pct
E-10 Evidence Evidence & Audit A single piece of evidence attached to a Case (screenshot, metadata JSON, image analysis result, NLP report). Has a SHA-256 hash for integrity. type, filePath, fileHash, capturedAt
E-11 BlockchainRecord Evidence & Audit An immutable Hyperledger Fabric transaction recording a case lifecycle event (EVIDENCE_LOCKED, TAKEDOWN_SENT, PLATFORM_RESPONSE, ESCALATED). txId, eventType, evidenceHash, blockNumber, channelName, timestamp
E-12 TakedownRequest Communication A formal delete/suspend request sent by the agency to a social media platform. Tracked for SLA compliance; linked to blockchain record. requestNumber, requestedAction, status, sentAt, slaDeadline, reminderCount
E-13 Notification Communication System-generated message to Citizens (status updates), Platforms (SLA reminders), or Officers (alerts). Sent via email/SMS/in-app. type, channel, recipient, message, sentAt, isDelivered
E-14 LegalReport Communication A court-admissible PDF dossier generated for escalated cases. Contains the full case timeline, evidence hashes, blockchain TX IDs and digital signature. reportNumber, digitalSignature, blockchainTxId, applicableSections[ ]
